# Booking System

A full-stack booking platform for service businesses such as salons,
barbershops, and clinics.

The project focuses on scheduling rules, availability, role-based access
control, and preventing conflicting reservations.

## Planned stack

- Node.js and TypeScript
- Express and PostgreSQL
- Prisma ORM
- JWT authentication
- Docker and GitHub Actions
- React admin dashboard (later phase)
